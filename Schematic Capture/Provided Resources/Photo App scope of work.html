Photo App scope of work

Scope of work for photo app-

Web site- Password protected with user access (Can this all be done from an app?)

Ability to create a client online and create projects under that client.

Ability to upload multiple individual job sheets to web site and associate each of them with a PDF schematic under a selected client/ project

App- Password protected with user access

Ability in the app to select a client / project /job sheet and see the job sheet data fields / contents in a list view.

Ability to edit job sheet field contents on the mobile device if errors are found or data is missing.

Ability to select the PDF schematic associated with the job sheet to see the schematic full screen.

Ability to select a component and have that open the Camera to take a picture.

Ability to use the same picture for multiple line items (bubble numbers)

Ability to retain high resolution unmarked image.

Ability to add arrows, circles, text to picture taken within the app

Ability for captured job sheets line items to be moved to a completed tab so you only see what is left.

Ability to select the completed tab to review what has been captured and modify if necessary.

Ability to have the pictures saved with the same name listed in the job sheet (1.jpg etc) There may be a duplication of 1.jpg across a project so the pictures need to be kept in separate folders (per job sheet)

Ability to quickly toggle between schematic / job sheet.

Ability to upload a new CSV file or update the existing in dropbox with all data (including what was added or changed offline)

Ability to capture all photos / edit job sheets info while offline

Ability to upload photos to dropbox when online in folders per job sheet

No Photo resize, keep full resolution on both pictures.

Must keep a copy of the photos on the device as well just in case.

Nice to have-

Ability to add additional components to a job sheet from the mobile device if items are missing.

Ability to have bubble numbers change color after a picture is taken for it

Ability to add a bubble to schematic (Some type of marker)
