Product Overview

Persona(s)

Problem

David has a lot of pictures that are now outdated, unusable and unorganized. He needs to find a better way to capture pictures of equipment he is responsible for and easily be able to associate the pictures to a specific machine and component.

Mission

David needs to find a better way to capture pictures of equipment he is responsible for and easily be able to associate the pictures to a specific machine and component. Often times, when a machine breaks down the maintenance crews will do whatever is necessary to get it back up and running. In many cases, that means changing out a component. Most of the time that change is never documented which can cause some safety issues. David needs a good way to update the job sheet data when he is taking pictures and notices something has been changed. He wants his maintenance guys to be able to safely troubleshoot the systems. When new people are hired he would like to be able to show them some pictures and discuss machine function before taking them out into the production area (for safety reasons) Currently that doesn't happen because it takes too long to find the pictures he is looking for because they are not named and related to a specific schematic.

Offer a fast way to document a large machine schematic is with pictures and annotations.

  • To be able to update old or outdated component information at the same time he is capturing the pictures.
  • To help people like David to be able to accomplish their documentation responsibilities as efficiently and accurately as possible, and to help educate the crews and help them stay safe while troubleshooting.
  • To be able to quickly capture pictures of critical machine components and have them automatically saved with a name that references the schematic for that machine.

Tech Stack Architecture

Example: Cloud Doc Searcher is a web only app built on a Postgres Database, and a Node Express back-end. The front-end is built on React, Redux and uses websockets and service workers for push notifications.

Schematic Capture is a Web / IOS app that is built on several technologies. Web: front end is built on JavaScript, React, Hooks, on a Node.js backend on a Postgres Database using Drop Box to store folders. IOS: Swift for front end. UX will utilize Figma for the design and user flow.
